# Roles

## Driver (Conductor) — MVP
- Registro, onboarding, KYC
- Recibir y gestionar viajes
- Pagos y earnings
- Emergencias (SOS)
- Rating de pasajeros

## Passenger (Pasajero) — Post-MVP
- Solicitar viajes
- Tracking en tiempo real
- Pagar con tarjeta/efectivo
- Rating de conductores

## Admin — Post-MVP
- Dashboard de gestión
- Aprobar/rechazar KYC
- Moderación de contenido
