# Pricing — Lifty

## Tarifa Conductor MVP

- **Tarifa base** (según zona y tipo de vehículo: auto/moto)
- **Distancia** calculada via Distance Matrix API
- **Split 80/20**: 80% conductor, 20% plataforma
- **Compensación por cancelación tardía** (>= 5 min esperando)
- **Retiro a CVU/CBU/alias** via Mercado Pago

### Pendiente
- Pricing dinámico / surge pricing
- Tarifas diferenciadas por distrito
